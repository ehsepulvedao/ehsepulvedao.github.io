Open Educational Resources

The following open educational resources correspond to laboratory guides and technical teaching materials developed for industrial automation, motion control and servomechanism training using the Allen-Bradley CompactLogix industrial platform.

2015

  1. UNAL
    Guía inicial para Control de Movimiento de Servomecanismos usando la Plataforma Industrial CompactLogix de Allen Bradley
    Edgar Hernando Sepúlveda-Oviedo, David Andrés Sáenz Ardila, Kevin Andrés Pulido, and 1 more author
    Sep 2015
    Abs DOI Bib

    This document introduces a comprehensive training framework for industrial motion control based on the Allen-Bradley CompactLogix platform. It presents the theoretical foundations of integrated industrial automation architectures, programmable automation controllers (PACs), industrial communication networks, and Studio 5000 software. The guide provides the conceptual basis required to design, configure, and implement coordinated multi-axis motion control systems in industrial environments, serving as the foundation for a complete hands-on training program in servo-driven automation systems.

  2. UNAL
    Guía 1: Configuración Hardware-Software I de III para Control de Movimiento de Servomecanismos usando la Plataforma Industrial CompactLogix de Allen Bradley
    Edgar Hernando Sepúlveda-Oviedo, David Andrés Sáenz Ardila, Kevin Andrés Pulido, and 1 more author
    May 2015
    Abs DOI Bib

    This guide focuses on the initial configuration of an industrial motion control system using Allen-Bradley CompactLogix controllers and Studio 5000 software. It covers project creation, controller configuration, communication setup, and input/output module identification. Through a three-axis industrial case study, participants learn the essential hardware and software configuration procedures required to build and deploy motion control applications in an industrial automation environment.

  3. UNAL
    Guía 2: Configuración Hardware-Software II de III para Control de Movimiento de Servomecanismos usando la Plataforma Industrial CompactLogix de Allen Bradley
    Edgar Hernando Sepúlveda-Oviedo, David Andrés Sáenz Ardila, Kevin Andrés Pulido, and 1 more author
    May 2015
    Abs DOI Bib

    This guide presents the configuration of industrial servo drives and motion control devices within the Allen-Bradley ecosystem. It addresses the integration of Kinetix servo drives, motion groups, motor assignment, axis configuration, and online controller commissioning. The document provides practical procedures for establishing communication between controllers, drives, and servomotors, enabling coordinated multi-axis motion control applications.

  4. UNAL
    Guía 3: Configuración Hardware-Software III de III para Control de Movimiento de Servomecanismos usando la Plataforma Industrial CompactLogix de Allen Bradley
    Edgar Hernando Sepúlveda-Oviedo, David Andrés Sáenz Ardila, Kevin Andrés Pulido, and 1 more author
    Jun 2015
    Abs DOI Bib

    This guide covers advanced commissioning tasks for industrial servo systems, including system sizing, startup validation, rotation direction configuration, and automatic controller tuning. Participants learn how to parameterize motion axes, perform hook-up tests, and optimize servo performance using the autotuning capabilities available in Studio 5000. The guide concludes the hardware and software configuration phase and prepares users for motion programming activities.

  5. UNAL
    Guía 4: Aspectos de programación de Movimiento I de III para Control de Movimiento de Servomecanismos usando la Plataforma Industrial CompactLogix de Allen Bradley
    Edgar Hernando Sepúlveda-Oviedo, David Andrés Sáenz Ardila, Kevin Andrés Pulido, and 1 more author
    Jun 2015
    Abs DOI Bib

    This guide introduces industrial motion programming using the Ladder language within Studio 5000. It presents the main PLC programming paradigms defined by IEC 61131 and focuses on the implementation of safety-related motion instructions, including emergency stop, shutdown, fault reset, and servo enable/disable functions. The guide establishes the programming foundations required to safely operate industrial motion systems and coordinate multiple servo axes.

  6. UNAL
    Guía 5: Aspectos de programación de Movimiento II de III para Control de Movimiento de Servomecanismos usando la Plataforma Industrial CompactLogix de Allen Bradley
    Edgar Hernando Sepúlveda-Oviedo, David Andrés Sáenz Ardila, Kevin Andrés Pulido, and 1 more author
    Jun 2015
    Abs DOI Bib

    This guide focuses on the implementation of motion control instructions for industrial servomechanisms. It covers homing procedures, jogging operations, point-to-point positioning, controlled stopping, and electronic gearing using Allen-Bradley motion function blocks. Through Ladder-based programming examples, participants develop the software architecture required for coordinated multi-axis motion applications.

  7. UNAL
    Guía 6: Aspectos de programación de Movimiento III de III para Control de Movimiento de Servomecanismos usando la Plataforma Industrial CompactLogix de Allen Bradley
    Edgar Hernando Sepúlveda-Oviedo, David Andrés Sáenz Ardila, Kevin Andrés Pulido, and 1 more author
    Jun 2015
    Abs DOI Bib

    This guide presents the complete implementation of a coordinated three-axis industrial motion application based on an automated box labeling and adhesive dispensing system. The document introduces comparison instructions, sequential logic design, state-based programming, and synchronized axis control. By integrating sensing, conveyor motion, and coordinated servo movements, the guide demonstrates the development of a realistic industrial automation application using Studio 5000 and Ladder programming.
